# VS Code Default Keyboard Shortcuts

## General

| Action | Windows | Mac |
|---|---|---|
| Command Palette | `Ctrl+Shift+P` / `F1` | `Cmd+Shift+P` / `F1` |
| Quick Open | `Ctrl+P` | `Cmd+P` |
| Settings | `Ctrl+,` | `Cmd+,` |
| Keyboard Shortcuts | `Ctrl+K Ctrl+S` | `Cmd+K Cmd+S` |

## Sidebar & Panels

| Action | Windows | Mac |
|---|---|---|
| Toggle Sidebar | `Ctrl+B` | `Cmd+B` |
| Toggle Terminal | `Ctrl+\`` | `Ctrl+\`` |
| Toggle Panel | `Ctrl+J` | `Cmd+J` |
| Explorer | `Ctrl+Shift+E` | `Cmd+Shift+E` |
| Search | `Ctrl+Shift+F` | `Cmd+Shift+F` |
| Source Control | `Ctrl+Shift+G` | `Cmd+Shift+G` |
| Run and Debug | `Ctrl+Shift+D` | `Cmd+Shift+D` |
| Extensions | `Ctrl+Shift+X` | `Cmd+Shift+X` |

## Editing

| Action | Windows | Mac |
|---|---|---|
| Select next occurrence | `Ctrl+D` | `Cmd+D` |
| Select all occurrences | `Ctrl+Shift+L` | `Cmd+Shift+L` |
| Move line up/down | `Alt+↑` / `Alt+↓` | `Opt+↑` / `Opt+↓` |
| Copy line up/down | `Shift+Alt+↑` / `Shift+Alt+↓` | `Shift+Opt+↑` / `Shift+Opt+↓` |
| Delete line | `Ctrl+Shift+K` | `Cmd+Shift+K` |
| Insert line below | `Ctrl+Enter` | `Cmd+Enter` |
| Insert line above | `Ctrl+Shift+Enter` | `Cmd+Shift+Enter` |
| Toggle line comment | `Ctrl+/` | `Cmd+/` |
| Toggle block comment | `Shift+Alt+A` | `Shift+Opt+A` |
| Format document | `Shift+Alt+F` | `Shift+Opt+F` |
| Format selection | `Ctrl+K Ctrl+F` | `Cmd+K Cmd+F` |
| Indent / Outdent | `Ctrl+]` / `Ctrl+[` | `Cmd+]` / `Cmd+[` |
| Go to beginning of line | `Home` | `Cmd+←` |
| Go to end of line | `End` | `Cmd+→` |
| Go to beginning of file | `Ctrl+Home` | `Cmd+↑` |
| Go to end of file | `Ctrl+End` | `Cmd+↓` |
| Undo cursor position | `Ctrl+U` | `Cmd+U` |

## Navigation

| Action | Windows | Mac |
|---|---|---|
| Go to Definition | `F12` | `F12` |
| Go to Implementation | `Ctrl+F12` | `Cmd+F12` |
| Find References | `Shift+F12` | `Shift+F12` |
| Peek Definition | `Alt+F12` | `Opt+F12` |
| Go to Line | `Ctrl+G` | `Ctrl+G` |
| Go to Symbol in File | `Ctrl+Shift+O` | `Cmd+Shift+O` |
| Go to Symbol in Workspace | `Ctrl+T` | `Cmd+T` |
| Navigate back | `Alt+←` | `Ctrl+-` |
| Navigate forward | `Alt+→` | `Ctrl+Shift+-` |
| Open file to the right | `Ctrl+Enter` (in Quick Open) | `Cmd+Enter` (in Quick Open) |

## Search & Replace

| Action | Windows | Mac |
|---|---|---|
| Find | `Ctrl+F` | `Cmd+F` |
| Replace | `Ctrl+H` | `Cmd+H` |
| Find in files | `Ctrl+Shift+F` | `Cmd+Shift+F` |
| Replace in files | `Ctrl+Shift+H` | `Cmd+Shift+H` |
| Find next / previous | `F3` / `Shift+F3` | `Cmd+G` / `Cmd+Shift+G` |

## Multi-cursor & Selection

| Action | Windows | Mac |
|---|---|---|
| Add cursor | `Alt+Click` | `Opt+Click` |
| Add cursor above/below | `Ctrl+Alt+↑` / `Ctrl+Alt+↓` | `Cmd+Opt+↑` / `Cmd+Opt+↓` |
| Column selection | `Shift+Alt+Drag` | `Shift+Opt+Drag` |
| Select current line | `Ctrl+L` | `Cmd+L` |
| Expand selection | `Shift+Alt+→` | `Ctrl+Shift+Cmd+→` |
